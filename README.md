# Address Book

## Project Description

The Address Book is a console-based application developed in C that helps users manage contact information. It allows users to add, search, edit, delete, save, and load contacts using file handling.

## Features

- Add a new contact
- Search for a contact
- Edit contact details
- Delete a contact
- Display all contacts
- Save contacts to a file
- Load contacts from a file

## Technologies Used

- C Programming
- Structures
- File Handling
- Functions
- Git
- GitHub
